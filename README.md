# Minecraft Server Properties Plugin

A Pelican plugin that provides a graphical editor for Minecraft `server.properties` files, allowing users to easily modify server settings through the web interface.

## Features

- Graphical interface for editing Minecraft server properties
- User-friendly form-based editing
- Integration with Pelican's server management panel

## Installation

1. Download the plugin files
2. Place the `minecraft-properties` folder in your Pelican `plugins` directory
3. Enable the plugin in your Pelican admin panel

## Usage

Once installed and enabled, users can access the server properties editor from the server management page in Pelican. The interface provides an easy way to modify common Minecraft server settings without directly editing the `server.properties` file.

## Requirements

- Pelican Panel
- Minecraft server integration
